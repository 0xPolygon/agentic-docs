---
title: "Get Token Id At Index For User"
description: "The getTokenIdAtIndexForUser method returns token id on supplied index for user."
---
The `getTokenIdAtIndexForUser` method returns token id on supplied index for user.

```js
const erc721Token = posClient.erc721(<token address>);

const result = await erc721Token.getTokenIdAtIndexForUser(<index>,<user address>);

```
