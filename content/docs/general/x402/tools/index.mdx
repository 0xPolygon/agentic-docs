# x402 Tools

Explore the available tools for interacting with x402-enabled APIs on Polygon.

## Available Tools

### [x402 Discovery Layer](/general/x402/tools/discovery-layer)
Learn how to discover and register x402-enabled resources using the Discovery API. Find APIs on Polygon networks through full-text search and filtering.

### [x402 Autopay](/general/x402/tools/autopay)
Browser extension that automatically handles x402 payments for web resources. Perfect for seamless browsing of paywall-protected content.

### [x402 CLI](/general/x402/tools/x402-cli)
Command-line interface for testing and interacting with x402 APIs. Ideal for developers and power users.
